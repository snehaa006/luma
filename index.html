Features

Designed for accessibility, built for everyone

Every feature is crafted to make the web usable through voice alone — no sighted assistance needed.

🎤

Wake Word Activation

Say "Hey Luma" to activate the assistant on any page. No need to click anything — it's always listening for you.

📖

Page Reader

Reads page content, headings, and links aloud. Navigate between headings and links with simple voice commands.

🎬

YouTube Control

Search videos, play, pause, skip, adjust volume, and go fullscreen — all hands-free on YouTube.

🧭

Page Navigation

Scroll, go back, jump to top or bottom, click elements by name, and open any website with a single command.

💬

Multi-Command Mode

Stays awake for 30 seconds after each command so you can chain multiple actions without repeating the wake word.

⌨️

Keyboard Shortcut

Press Ctrl+Shift+S (or Cmd+Shift+S on Mac) to instantly toggle the assistant without saying the wake word.

Install

Get started in under a minute

Luma Voice Extension runs on Chrome, Edge, Brave, and any Chromium-based browser.

Download ZIP from GitHub

Download the extension

Click the download button above. You'll get a .zip file containing all extension files.

Unzip the folder

Extract the ZIP file to a permanent location on your computer. Don't delete this folder — Chrome needs it.

Open the extensions page

Open Chrome and go to chrome://extensions in the address bar.

Enable Developer Mode

In the top-right corner, toggle on Developer mode.

Load the extension

Click "Load unpacked" then select the unzipped folder. Luma will appear in your extensions list.

Allow microphone access

Visit any website and allow microphone access when prompted. Say "Hey Luma" — you're ready.

FAQ

Frequently asked questions

Quick answers to common questions about the Luma Voice Extension.

Which browsers are supported?

Chrome, Edge, Brave, Opera, and any Chromium-based browser with the Web Speech API. Firefox and Safari are not yet supported.

Is it free?

Yes, completely free and open source. No accounts, no subscriptions, no data collection.

Does it send my voice data anywhere?

Luma uses your browser's built-in speech recognition (Web Speech API). Voice processing is handled by your browser — no audio is transmitted or stored.

Can I use it without saying the wake word?

Yes. Press Ctrl+Shift+S (or Cmd+Shift+S on Mac) to activate the assistant instantly.

Why does it ask for microphone permission on every site?

Chrome requires per-site microphone permission for security. You can allow it permanently in Chrome's site settings.

The assistant isn't hearing me. What should I do?

Check microphone access in the address bar lock icon. Try refreshing the page. If using Bluetooth, ensure it's set as the default input device.
